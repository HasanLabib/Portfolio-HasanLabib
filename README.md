# S. M. Hasan Imtiaz Labib Portfolio - React Version

This is a React conversion of Jensen Omega's portfolio website, maintaining the exact same styling and design while converting from static HTML to a dynamic React application.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Dark Mode Support**: Automatic dark mode based on system preferences
- **Modern Styling**: Uses Tailwind CSS for consistent and modern styling
- **Component-Based Architecture**: Modular React components for easy maintenance
- **Routing**: React Router for navigation between pages
- **Interactive Elements**: Hover effects, animations, and smooth transitions

## Project Structure

```
src/
├── components/
│   ├── Navigation.js      # Main navigation bar
│   ├── Hero.js           # Hero section with profile image
│   ├── TechStack.js      # Technology stack banner
│   ├── About.js          # About section with experience/education
│   ├── Skills.js         # Technical skills grid
│   ├── Projects.js       # Projects showcase
│   ├── Contact.js        # Contact form and social links
│   └── Footer.js         # Footer component
├── pages/
│   ├── Home.js           # Main portfolio page
│   └── ProjectDetails.js # Project details page
├── App.js                # Main app component with routing
└── index.js              # React entry point
```

## Technologies Used

- **React 18**: Modern React with hooks
- **React Router DOM**: Client-side routing
- **Tailwind CSS**: Utility-first CSS framework
- **Material Icons**: Google Material Icons
- **Font Awesome**: Icon library

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm start
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```
